---
title: "شروع سریع"
description: "راه‌اندازی محیط توسعه OrgaFlow"
---

1. مخزن را کلون کنید.
2. `pnpm i` را اجرا کنید.
3. فایل `.env.example` را به `.env` کپی کنید.
4. `pnpm docker:up` برای اجرای سرویس‌های وابسته.
5. `pnpm db:migrate && pnpm db:seed`.
6. `pnpm dev`.

> نکته: برای تست وب‌هوک‌ها می‌توانید از سرویس Ngrok استفاده کنید.
